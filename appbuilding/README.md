# 大学生健康守护 App

## 项目概述

大学生健康守护 App 是一款专为大学生设计的健康管理应用，旨在帮助学生监测和改善睡眠质量、管理课程时间、减少熬夜行为，并提供健康提醒功能。通过数据可视化和智能分析，帮助学生建立健康的生活习惯。

## 功能特点

### 核心功能
- **健康提醒完成率**：展示每日健康提醒的完成情况，通过环形图表直观呈现
- **今日课程**：显示当天的课程安排，包括课程名称、时间和教室信息
- **睡眠分析**：
  - 一周睡眠时长趋势折线图
  - 有效/无效熬夜时长对比柱状图
  - 熬夜类型分布饼图
- **课程时长统计**：展示一周内每日课程时长的分布情况
- **亮屏时长统计**：监控一周内每日手机亮屏时长，帮助用户了解屏幕使用习惯

### 技术特点
- **响应式设计**：适配不同屏幕尺寸，在手机和电脑上都能良好显示
- **数据可视化**：使用 Chart.js 实现丰富的数据图表展示
- **模块化架构**：采用 Vue 3 组件化开发，代码结构清晰
- **实时数据**：通过 API 接口获取实时数据（目前使用模拟数据）

## 技术栈

### 前端
- **框架**：Vue 3 + Vite
- **路由**：Vue Router
- **数据可视化**：Chart.js
- **样式**：CSS3 (使用 CSS 变量实现主题管理)

### 后端
- **语言**：Java
- **框架**：Spring Boot
- **数据库**：SQLite
- **API**：RESTful API

## 安装指南

### 前端安装
1. 进入前端目录
   ```bash
   cd frontend
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 启动开发服务器
   ```bash
   npm run dev
   ```

4. 构建生产版本
   ```bash
   npm run build
   ```

### 后端安装
1. 进入后端目录
   ```bash
   cd backend
   ```

2. 构建项目
   ```bash
   mvn clean package
   ```

3. 运行应用
   ```bash
   java -jar target/health-guard-1.0.jar
   ```

## 使用说明

### 首页（健康仪表盘）
- **健康提醒完成率**：显示当天健康提醒的完成情况
- **今日课程**：查看当天的课程安排
- **睡眠分析**：查看一周睡眠时长趋势、熬夜对比和类型分布
- **课程时长统计**：了解一周内每日课程时长
- **亮屏时长统计**：监控手机屏幕使用时间

### 用户管理
- 查看和编辑个人信息
- 管理账户设置

### 课程管理
- 查看课程表
- 导入和导出课程

### 睡眠分析
- 查看详细的睡眠记录
- 评价睡眠质量
- 查看亮屏时长统计

### 提醒管理
- 设置健康提醒
- 查看提醒历史

## 数据说明

目前项目使用模拟数据进行展示，实际部署时需要连接后端 API 获取真实数据。模拟数据包括：

- 一周睡眠时长数据
- 有效/无效熬夜时长数据
- 熬夜类型分布数据
- 每日课程时长数据
- 亮屏时长数据
