# 大学生健康守护 App - 前端

## 项目概述

大学生健康守护 App 是一个针对大学生健康管理的应用，前端部分使用 Vue 3 + Vite 构建，提供了直观、易用的用户界面，帮助学生管理健康数据、课程表、睡眠质量和提醒计划。

## 技术栈

- **框架**：Vue 3
- **构建工具**：Vite
- **路由**：Vue Router
- **HTTP 客户端**：Axios
- **样式**：CSS3

## 项目结构

```
frontend/
├── src/
│   ├── components/       # 通用组件
│   │   ├── Navbar.vue    # 底部导航栏
│   │   └── Footer.vue    # 页脚（已移至提醒管理页面）
│   ├── router/          # 路由配置
│   │   └── index.js     # 路由定义
│   ├── services/        # API 服务
│   │   └── api.js       # API 调用封装
│   ├── views/           # 页面组件
│   │   ├── Home.vue     # 首页
│   │   ├── User.vue      # 用户管理
│   │   ├── Schedule.vue # 课表管理
│   │   ├── Sleep.vue     # 睡眠分析
│   │   └── Reminders.vue # 提醒管理
│   ├── App.vue          # 根组件
│   └── main.js          # 入口文件
├── index.html           # HTML 模板
├── package.json         # 依赖配置
└── vite.config.js       # Vite 配置
```

## 安装和运行

### 前提条件

- Node.js 14.0+
- npm 6.0+

### 安装依赖

```bash
cd frontend
npm install
```

### 运行开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:3000/` 运行。

### 构建生产版本

```bash
npm run build
```

构建产物将生成在 `dist` 目录。

## 主要功能

1. **首页**：显示健康状态概览、今日课程和近期提醒
2. **用户管理**：创建用户、提交健康档案
3. **课表管理**：可视化定制课表、导入课表、查看今日课程
4. **睡眠分析**：查看熬夜记录、提交睡眠质量评价
5. **提醒管理**：生成提醒计划、查看待处理提醒、响应提醒

## 核心功能说明

### 课表定制

课表管理页面提供了可视化的课表定制功能：
- 以周为单位的网格布局
- 点击空白格子添加课程
- 填写课程详情（名称、教室、时间、位置等）
- 支持编辑和删除课程
- 一键提交课表到后端

### 健康数据管理

- 记录亮屏事件和灭屏事件
- 分析熬夜评分
- 提交睡眠质量评价
- 生成健康提醒

### 提醒系统

- 根据熬夜记录和课程表生成每日提醒计划
- 支持晨起健康提醒、出发提醒、路口提醒等
- 响应提醒（收到、关闭、推迟）

## API 调用

前端通过 `services/api.js` 封装了与后端的所有 API 调用，包括：

- 用户管理相关接口
- 课表管理相关接口
- 数据采集相关接口
- 睡眠分析相关接口
- 提醒调度相关接口
- 外部 API 相关接口

## 响应式设计

应用采用响应式设计，适配不同屏幕尺寸，包括桌面端、平板和移动设备。
